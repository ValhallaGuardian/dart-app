Framework: React 19 – najnowszy standard rynkowy, gwarantujący wydajność i modułowość.
Język: TypeScript – silne typowanie danych. Zapobiega błędom typu "undefined" i gwarantuje, że dane wyświetlane graczowi są poprawne.
Build Tool: Vite – błyskawiczne ładowanie aplikacji.
Styling: Tailwind CSS (v3.4) – podejście "Mobile First". Aplikacja jest responsywna (RWD) i zoptymalizowana pod ekrany dotykowe (duże przyciski, ciemny motyw oszczędzający baterię/oczy).
Komunikacja: Socket.io Client – obsługa połączenia w czasie rzeczywistym (Real-time).
Architektura: PWA (Progressive Web App) – aplikację można zainstalować na telefonie, działa pełnoekranowo (bez paska adresu przeglądarki) i zachowuje się jak natywna aplikacja.
Routing: React Router – obsługa nawigacji (Logowanie -> Lobby -> Gra -> Profil).
